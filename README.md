# timeleaderboard-app

Leaderboard for a wakatime-like app, with features for uploading time-tracking data from ActivityWatch to a public or private group leaderboard in different categories. User accounts should be able to connect to GitHub.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App-Dev/timeleaderboard-app.git
cd timeleaderboard-app
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
